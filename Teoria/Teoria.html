// Configuracion del BACK 1-Primero instalamos Node.js, a traves del comando "npm init -y" 2-Instalamos json-server a traves del comando "npm i json-server" 3-Creamos el archivo para utilizar la fake API, en este caso sera el archivo "db.json" 4-Se crea la tabla de usuraios en el archivo mencionado anteriormente 5-Se configura el archivo "package.json", se borra dentro del scripts, la linea test y se reemplaza por "start" 6-Luego se inicia el back con el comando "npm start" //Configuracion del FRONT 1-Instalamos React a traves del comando "npx create-react-app login-front" 2-Iniciamos el proyecto a traves del comando "npm start" 3-Instalamos a traves del comando "npm i react-router-dom" para poder utilizar las rutas 4-Se importan los componentes a usar "import {BrowserRouter, Switch, Route} from 'react-router-dom';" 5-Creacion de carpetas "CSS" para las hojas de estilo, "pages" para las paginas que seran el LOGIN y el MENU PRINCIPAL y por ultimo "routes" para las rutas que vamos a crear 6-Instalacion de bootstrap, axios, md5 y universal-cookie
